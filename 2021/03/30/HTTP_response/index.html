HTTP_response

HTTP响应状态码：

  • 1xx 信息，服务器收到请求，需要请求者继续执行操作

  • 2xx 响应成功

  • 3xx 重定向，需要进一步的操作以完成请求

  • 4xx 客户端错误

  • 5xx 服务端错误

1. HTTP 响应头（格式：头名称：值）

  • Content-Type 服务器告诉客户端本次响应体数据格式以及编码格式。

  • Content-Length 返回的内容长度

  • Content-dispostion 服务器告诉客户端以什么格式打开响应体数据

    • in-line :默认值

    • attachment :以附件形式打开响应体

2. 响应空行

3. 响应体
